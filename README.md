# Problem-Solver
A framework for solving some kinds of problems! Including:  
* Classical tree-search
  * DFS
  * Evolutionary DFS  
  * BFS  
  * Bidirectional BFS  
  * Backtracking  
* Intelligent inexact problems:
  * HillClimbing
    * Simple
    * First Choice
    * Random Choice
    * Random Restart
  * Genetic Algorithm
